# Introdução a Classes em Java

Em Java, uma **classe** é uma estrutura utilizada para definir um modelo ou estrutura que representa um objeto do mundo real dentro do programa. Ela descreve quais dados o objeto possui e quais comportamentos ele pode executar.

Uma classe normalmente é composta por dois elementos principais:

- **Atributos**: representam as características ou dados do objeto.
- **Métodos**: representam as ações ou comportamentos que o objeto pode executar.

A partir de uma classe é possível criar **objetos**. Esse processo é chamado de **instanciação**. Cada objeto criado a partir de uma classe possui seus próprios valores para os atributos definidos.

Por exemplo, uma classe `Pessoa` pode possuir atributos como `nome` e `idade`, e métodos como `falar()` ou `andar()`.

A programação orientada a objetos (POO) organiza o código em classes e objetos para tornar os sistemas mais organizados, reutilizáveis e fáceis de manter.
