# EstudosJava

Repositório dedicado ao aprendizado e prática da linguagem **Java**, indo do nível **básico ao avançado**. Aqui estão reunidos exemplos, exercícios e pequenos projetos criados durante os estudos, com o objetivo de consolidar conceitos fundamentais e evoluir gradualmente para tópicos mais complexos da linguagem e do ecossistema Java.

## Objetivo

Este repositório foi criado para:

- Praticar e registrar a evolução no aprendizado de Java
- Reforçar conceitos fundamentais da linguagem
- Explorar recursos mais avançados e boas práticas
- Servir como material de consulta futura

## Conteúdos abordados

Ao longo do repositório são abordados temas como:

- Sintaxe básica do Java
- Variáveis e tipos de dados
- Estruturas condicionais (if, switch)
- Estruturas de repetição (for, while, do-while)
- Métodos
- Arrays e coleções
- Programação Orientada a Objetos (POO)
- Encapsulamento, herança e polimorfismo
- Tratamento de exceções
- Streams e programação funcional
- Boas práticas e organização de código
- Pequenos projetos para aplicação prática

## Estrutura do repositório

O conteúdo está organizado em pastas de acordo com os temas estudados, geralmente seguindo uma progressão de dificuldade:

/basico  
/intermediario  
/avancado  
/projetos  

Cada pasta contém exemplos de código, exercícios resolvidos e experimentos realizados durante os estudos.

## Tecnologias

- Java
- JDK
- IDE de desenvolvimento (IntelliJ, Eclipse ou VS Code)

## Observação

Este repositório faz parte de um processo contínuo de aprendizado. O conteúdo pode ser atualizado, reorganizado ou expandido conforme novos tópicos forem estudados.
